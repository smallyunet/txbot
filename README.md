
## txbot

Get market signals from https://www.bi123.co and auto-make orders in Binance.

## Install

```
pip install python-binance
pip install -r requirements.txt
```

## Configuration

`config.py` format:

```
EMAIL = ''
PASSWORD = ''
SERVER = ''
PORT = 0

telegram_api_token = ''
telegram_chat_id = ''

biance_api_key = ''
biance_secrect_key = ''
```
